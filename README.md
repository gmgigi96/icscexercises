# Modern I/O & Storage Systems Lab (iCSC 2026)

This repository contains the C source code, Makefiles, and LaTeX instructions for the practical exercises on Linux I/O performance, zero-copy networking, and `io_uring`.

These labs are designed to demonstrate the "Syscall Tax," the "Double Copy" problem, and how modern storage nodes (like those in the WLCG) achieve maximum throughput.
